\section{Preprocessing}
% ---------------------------------------------------------------------

The following preprocessing steps were applied to all text sources:
\begin{itemize}
	\item Basic string manipulation: Text was converted to lowercase. URLs, @mentions, hashtag symbols, non-alphabetic characters, and excess whitespace were removed.
	\item Tokenization \& filtering: Tokens were filtered using NLTK English stopwords, with removal of tokens shorter than three characters and URL-slug-like strings. Language detection via langdetect retained only English documents.
	\item Stemming: For Bluesky TF-IDF analysis, SnowballStemmer was additionally applied. No stemming or lemmatization was used elsewhere.
	\item Buzz labeling: Documents were classified as TrumpBuzz, HarrisBuzz, or ElectionBuzz via whole-word keyword matching. The highest-scoring label was assigned provided it led by at least one match; ties defaulted to ElectionBuzz.
\end{itemize}
